# Containerized 3-Tier Student Management System

A Dockerized microservices-based Student Management System developed for the Virtual System & Services Lab course.

## Architecture

The system follows a 3-tier architecture:

- Frontend: Nginx + HTML/CSS
- Backend: Node.js + Express.js
- Database: Microsoft SQL Server 2019
- Orchestration: Docker Compose

## Features

- Docker containerization
- OS-level virtualization
- Microservices architecture
- REST API integration
- SQL database persistence
- Docker networking
- Infrastructure as Code (IaC)

## Tech Stack

- Docker
- Docker Compose
- Node.js
- Express.js
- Nginx
- Microsoft SQL Server
- HTML/CSS/JavaScript

## Run the Project

```bash
docker compose up --build
```

Frontend:
```text
http://localhost:8080
```

Backend API:
```text
http://localhost:5000
```
